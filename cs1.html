Computer Science I

Offered by Berrien County Mathematics & Science Center

What are HTML and its related languages?


Computer Science I is a class about webpage design. We learn about HTML, CSS, Javascript, and the best practices when using these coding languages. HTML, or HyperText Markup Language, is the coding language that is the skeleton of webpages. All websites on the internet contain HTML, which tells the browser the text and content of the page and the order to display it in.

CSS, or Cascading Style Sheets, make the web more visually appealing. CSS can be used to color text, shapes, and sections of the page. CSS can also be used for many other things, including positions, fonts, and borders.

Javascript is a the language that makes the internet interactive. Whenever something on the page responds to your actions or changes in appearance, chances are, the page is using javascript. Javascript is incredibly useful because it allows your page to respond to clicks, scrolling, and hovers in any way you could imagine.

Goals of the class


The goals of the Computer Science I class were:

  • To introduce us to the world of web developing
  • To teach us how to design and build websites
  • To enable us to build webpages on our own

Activities


During this class, we completed various activities pertaining to web development, including designing webpages from scratch, programming interesting javascript programs, and cloning popular websites. My favorite were the javascript programs, that included:

  • A web calculator
  • A set of buttons that modified the background color of the page
  • A reaction tester

Review


Overall, I felt that this course was a good introduction to web development. I learned how to develop professional-looking websites from scratch, in a single year. Sure, I knew a bit of HTML and CSS before this course, but it is nothing compared to what I have learned.
